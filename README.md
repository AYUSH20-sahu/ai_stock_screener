# AI Stock Screener

Production-ready AI-powered Indian stock screener.

## Milestone 1: Initialize

This milestone establishes the initial workspace structure for the frontend and backend, including:

- Next.js 15 + React 19 + TypeScript + Tailwind CSS + shadcn/ui setup
- Express + TypeScript + Node development tooling setup
- Environment files and package scripts
- Initial health endpoint for the backend

## Scripts

- Frontend: npm run dev --workspace frontend
- Backend: npm run dev --workspace backend
- Full stack: npm run dev
